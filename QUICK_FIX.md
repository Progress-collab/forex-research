# Быстрое решение: Что делать если Git LFS завис

## Ситуация
Установка Git LFS зависла или не работает, но нужно закинуть данные на GitHub.

## Быстрое решение (без Git LFS)

Если Git LFS не работает, можно добавить данные обычным способом:

```bash
# 1. Добавить данные в git (обычным способом, без LFS)
git add data/v1/curated/ctrader/*.parquet

# 2. Закоммитить
git commit -m "Add forex data files"

# 3. Запушить
git push
```

⚠️ **Внимание**: Это может быть медленно для больших файлов, но сработает!

## Или продолжить без данных в git

Можно продолжить работу, а данные добавить позже:

```bash
# Закоммитить код без данных
git add .
git commit -m "Update project - data will be added later"
git push
```

## После этого

Когда Git LFS заработает, можно будет:
1. Установить Git LFS
2. Переместить данные в LFS: `git lfs migrate import --include="*.parquet"`

## Что выбрать?

- **Если данные небольшие (< 100MB каждый файл)**: Используйте обычный git add
- **Если данные большие**: Попробуйте переустановить Git LFS или продолжите без данных пока
