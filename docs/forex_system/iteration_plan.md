# План улучшений и масштабирования

## Итерация 1 (0–1 месяц)
- Стабилизировать Prefect-потоки: добавить ретраи, оповещения о неудачных задачах.
- Автоматизировать выгрузку макроданных (ставки ФРС, PMI) для факторных фильтров.
- Провести параметрическую оптимизацию стратегий через grid search (MLflow).

## Итерация 2 (1–2 месяц)
- Добавить фактор ликвидности и реал-тайм спред для динамического sizing.
- Расширить набор инструментов: добавить пары с экзотикой (USDRUB через форекс-контракт FXPro) при условии приемлемых издержек.
- Внедрить адаптивные фильтры режима рынка (кластеризация по волатильности).

## Итерация 3 (2–3 месяц)
- Рассмотреть портфельный оптимизатор (risk parity) на live-результатах.
- Перенести execution в отказоустойчивый контур (двойные инстансы, health-check).
- Настроить автоматическое переобучение/валидацию стратегий и CI/CD пайплайн.

## Контрольные точки
- Еженедельные ретроспективы, обновление чек-листов и мониторинга.
- Постоянная оценка затрат/спредов у FXPro, пересмотр notional при изменениях.
